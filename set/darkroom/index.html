This is a 1.5 m² growing tent converted into a small darkroom in my 7m² office room ;) So far it works pretty well for wet plate. Next days I will pimp it up a bit. It has a microphone built in where I can set the lights to red, white, on and out via voice control. Also I can set timers via voice. This is controlled via an rasberry pi with rhasspy as voice recognition framework. Cloudfree. Not Alexa, no google.
